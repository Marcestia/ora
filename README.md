# ora

Site statique pour Oxysouffle R' Aquitaine (ORA).

## Lancer en local

Ouvrir `index.html` dans un navigateur.

## Structure

- `index.html` : page d'accueil
- `pour-qui.html` : public concerné
- `activites.html` : activités
- `pourquoi-nous-rejoindre.html` : bénéfices de l'association
- `adhesion.html` : demande d'adhésion
- `contact.html` : coordonnées et formulaire
- `styles.css` : styles globaux
- `script.js` : interactions front
- `assets/` : images du site
